Este proyecto se ha construido con .net core 5, por lo tanto:

- descarga la versión 2019 de visual studio
- clona el repositorio
- espera a que se restauren las dependencias
- cambia la ruta de archivo de destino (excel)
- cambia la ruta de carpeta de origen
- ejecuta

Puedes agregar mas tipos de archivos en la carpeta para ello:

- revisa como se llaman los archivos a cargar y en que hoja se van a cargar
- agrega una condición más en los nombres de archivos
- crea la hoja y agrega el nombre en la función leer archivo
- ejecuta

para dudas y/o comentarios envía un comentario el dueño del repositorio

¡Saludos!
